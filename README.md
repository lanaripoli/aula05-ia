## 📋 Práticas realizadas

### Prática 1 — IA como Copiloto
- Geração de função para verificar número primo em Python
- Explicação linha a linha do código gerado
- Versão otimizada com clean code e boas práticas
- Arquivo markdown com a explicação completa

### Prática 2 — Refatoração
- Código original mal estruturado fornecido para a IA
- Refatoração com nomes descritivos, funções com responsabilidade única
- Análise das melhorias: nomenclatura, estrutura, documentação

### Prática 3 — Debug com IA
- Código com 4 erros propositais (sintaxe, lógica, escopo, nome)
- Identificação, explicação e correção de cada erro

### Prática 4 — IA Explicando Código
- Código da Prática 2 explicado linha por linha pela IA
- Validação da capacidade interpretativa do assistente
